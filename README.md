# Dotfiles

Set up a new system, and maintain current configurations. Symlinking to `$HOME` is managed by [Dotbot][dotbot].

```bash
git clone --recurse-submodules git@github.com:rajitbanerjee/dotfiles ~/.dotfiles
cd ~/.dotfiles && ./install
```

```bash
# setup-system-ubuntu
setup-system-mac
setup-coc-nvim
nvim

# install-mac-apps
```
