# Setting up Jupyter Lab in WSL
1. `conda install -c conda-forge jupyterlab`
2.  Edit ~/.jupyter/jupyter_notebook_config.json:
```
{
  "NotebookApp": {
    "nbserver_extensions": {
      "jupyter_http_over_ws": true
    },
    "use_redirect_file": false
  }
}
```
3. Now, `jupyter lab` works in WSL terminal.
